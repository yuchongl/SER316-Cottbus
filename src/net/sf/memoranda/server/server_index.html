SER316 memoranda project HTTP server. Task #76

Storage for memoranda notes

This server uses HTTP query as parameters. It currently supports two operations: save note and read note. For notes operation, the format of the query is listed below:

/mem?op=[notes_op]&action=[save | load]&title=[...]&content=[...]

Note that the title of the note is used to identify it.

Saving a note

Example: GET http://104.236.193.225:7501/mem?op=notes_op&action=save&title=test&content=test_content

responses "OK" on success.

Reading a note

Example: GET http://104.236.193.225:7501/mem?op=notes_op&action=load&title=test

responses "no such note" if the note with the specified title does not exist.

responses the serialized content of the note if it is found.

Example response: 2;5,6,title,test23;7,12,content,test_content;

To deserialize the note, use the following code:

NoteStorage ns = new NoteStorage();
ns.deserialize(str);

Note list

Example: GET http://104.236.193.225:7501/mem?op=list

Server management

http://104.236.193.225:7501/mem?op=manage
